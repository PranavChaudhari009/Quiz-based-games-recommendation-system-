# Quiz-based-games-recommendation-system-
🎮 Quiz-Based Game Recommendation System

A web-based game recommendation system that suggests 4–5 games based on user quiz responses such as difficulty, multiplayer preference, platform, and playtime.
The system uses Machine Learning (scikit-learn) for genre prediction and Flask for backend integration.

📌 Features

🎯 Quiz-based input system

🤖 Machine Learning model for genre prediction

🎮 Recommends multiple games (not just one)

🖼️ Game images displayed as cards

▶️ Clickable images open YouTube game trailers

🔁 “Try Again” option to retake the quiz

💻 Clean UI using HTML & CSS



🛠️ Tech Stack

Frontend

-HTML5

-CSS3

Backend

-Python

-Flask

Machine Learning

-scikit-learn

-pandas
